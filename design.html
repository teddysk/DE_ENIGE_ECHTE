User testing

I asked 5 test users for my user testing report. Everyone of my test users did the test on their phone. I did a briefing before they did their user testing, and I made sure that they were well informed about the purpose and goal of the testing. 

 Most of the people reacted positive, it is an easy going website. Clear and not difficult at all. The feedback that I got is mostly about the “search” option which is not working, and that the content of the webpage is not finished. 

 My own recommendation for the website, I think the biggest thing that needs to be improved is the professionality of the website and the diversity in image using. Most of the things work and it’s pretty clear overall, but one thing that needs to be worked out is the “search” option. All of them did the test on their phone, which is also an improvement for myself next time, to make sure that there is a difference in devises that are being used for my user test report.
